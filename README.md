# OurTube

Youtube Video Downloader using yt-dlp, ffmpeg and Qt6.

# Build Instructions

## Windows

### Prerequisites
- [7zip](https://www.7-zip.org/)
- [Qt6 for MSVC](https://www.qt.io/download-qt-installer-oss)
- [Visual Studio 2022/2026](https://visualstudio.microsoft.com/vs/community/)

### Instructions



## Arch Linux

### Prerequisites
```sh
$ sudo pacman -S --needed qt6-base yt-dlp ffmpeg
```

### Instructions
```sh
$ make
```
